    Install the builded image

    Replace the ABCHOST var set to localhost by default to your own server ip address

    ABCHOST=localhost
curl --output ksquares.d.3.0.json https://raw.githubusercontent.com/abcdesktopio/oc.apps/main/ksquares.d.3.0.json
curl -X PUT -H 'Content-Type: text/javascript' http://$ABCHOST:30443/API/manager/image -d @ksquares.d.3.0.json

    Rebuild the image manually

    Download the Dockerfile manually

    Dockerfile for application kSquares

    wget https://raw.githubusercontent.com/abcdesktopio/oc.apps/main/kSquares.d

    build the Dockerfile to create a container image

    docker build --build-arg TAG=3.0 -f kSquares.d -t kSquares .

    Install the new image

    If you are using containerd as container runtime, use the ctr command line

    If you are not running this bash command on your abcdesktop node Replace the ABCHOST variable set to localhost by default to your own server ip address

    ABCHOST=localhost
docker inspect kSquares > kSquares.json
docker image save kSquares -o kSquares.tar
ctr -n k8s.io images import kSquares.tar
curl -X PUT -H 'Content-Type: text/javascript' http://$ABCHOST:30443/API/manager/image -d @kSquares.json
