    Install the builded image

    Replace the ABCHOST var set to localhost by default to your own server ip address

    ABCHOST=localhost
curl --output Leocad.json https://raw.githubusercontent.com/abcdesktopio/oc.apps/main/Leocad.d.3.0.json
curl -X PUT -H 'Content-Type: text/javascript' http://$ABCHOST:30443/API/manager/image -d @Leocad.json

    Rebuild the image manually

    Download the Dockerfile manually

    Dockerfile for application Leocad

    wget https://raw.githubusercontent.com/abcdesktopio/oc.apps/main/Leocad.d

    build the Dockerfile to create a container image

    docker build --build-arg TAG=3.0 -f Leocad.d -t Leocad .

    Install the new image

    If you are using containerd as container runtime, use the ctr command line

    If you are not running this bash command on your abcdesktop node Replace the ABCHOST variable set to localhost by default to your own server ip address

    ABCHOST=localhost
docker inspect Leocad > Leocad.json
docker image save Leocad -o Leocad.tar
ctr -n k8s.io images import Leocad.tar
curl -X PUT -H 'Content-Type: text/javascript' http://$ABCHOST:30443/API/manager/image -d @Leocad.json
