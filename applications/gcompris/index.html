    Install the builded image

    Replace the ABCHOST var set to localhost by default to your own server ip address

    ABCHOST=localhost
curl --output gcompris.d.3.0.json https://raw.githubusercontent.com/abcdesktopio/oc.apps/main/gcompris.d.3.0.json
curl -X PUT -H 'Content-Type: text/javascript' http://$ABCHOST:30443/API/manager/image -d @gcompris.d.3.0.json

    Download the Dockerfile manually

    Dockerfile for application GCompris

    wget https://raw.githubusercontent.com/abcdesktopio/oc.apps/main/GCompris.d

    build the Dockerfile to create a container image

    docker build --build-arg TAG=3.0 -f GCompris.d -t GCompris .

    If you are using containerd as container runtime, use the ctr command line
