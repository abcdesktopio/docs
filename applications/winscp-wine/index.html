    WM_CLASS

    winscp.exe.Wine

    The WM_CLASS property (of type STRING without control characters) contains two consecutive null-terminated strings. These specify the Instance and Class names to be used by both the client and the window manager for looking up resources for the application or as identifying information. to get the WM_CLASS property of an application, use the command line wmctrl -lx

    PRE run command

    PRE run command are run before the package install command

    ENV WINEARCH=win64
ENV WINEDLLOVERRIDES="mscoree,mshtml="
USER $BUSER
RUN wineboot --init
RUN curl -Ls -o /tmp/winscp553.zip http://winscp.net/download/winscp553.zip && unzip /tmp/winscp553.zip -d /composer/bin/ && rm /tmp/winscp553.zip
RUN echo disable > $WINEPREFIX/.update-timestamp
COPY --chown=$BUSER:$BUSER user.reg system.reg /composer/.wine/

    Install the builded image

    Replace the ABCHOST var set to localhost by default to your own server ip address

    ABCHOST=localhost
curl --output winscp-wine.d.3.0.json https://raw.githubusercontent.com/abcdesktopio/oc.apps/main/winscp-wine.d.3.0.json
curl -X PUT -H 'Content-Type: text/javascript' http://$ABCHOST:30443/API/manager/image -d @winscp-wine.d.3.0.json

    Rebuild the image manually

    Download the Dockerfile manually

    Dockerfile for application winscp-wine

    wget https://raw.githubusercontent.com/abcdesktopio/oc.apps/main/winscp-wine.d

    build the Dockerfile to create a container image

    docker build --build-arg TAG=3.0 -f winscp-wine.d -t winscp-wine .

    Install the new image

    If you are using containerd as container runtime, use the ctr command line

    If you are not running this bash command on your abcdesktop node Replace the ABCHOST variable set to localhost by default to your own server ip address

    ABCHOST=localhost
docker inspect winscp-wine > winscp-wine.json
docker image save winscp-wine -o winscp-wine.tar
ctr -n k8s.io images import winscp-wine.tar
curl -X PUT -H 'Content-Type: text/javascript' http://$ABCHOST:30443/API/manager/image -d @winscp-wine.json
